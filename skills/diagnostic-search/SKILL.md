# Diagnostic Search Skill

Use this skill to search, organize, export, import, or inspect debug history.

## Workflow

1. Search with a precise query and filters.
2. Save recurring queries when useful.
3. Use export/import only for backup or migration workflows.
4. Inspect recorder diagnostics when the recorder itself may be unhealthy.
5. Treat deleting session history as destructive and require explicit approval.
